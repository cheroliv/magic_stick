== Guide de demarrage rapide

Ce guide vous permet d'etre operationnel avec Magic Stick en quelques minutes.

=== Telecharger l'ISO

L'ISO est construite automatiquement par GitHub Actions. Pas besoin de builder localement.

L'ISO est disponible sur SourceForge (ISO brute, prete a flasher) :

* https://sourceforge.net/projects/magic-stick/files/[magic-stick sur SourceForge]

Pour le guide complet avec balenaEtcher et dd, consultez link:flash-guide.html[Guide d'installation et flashage USB].

=== Etape 1 : Flasher la cle USB

avec **balenaEtcher** (recommande) :

. Telechargez https://balena.io/etcher[balenaEtcher]
. **Flash from file** → selectionnez `magic_stick_0.1.8.iso`
. **Select target** → votre cle USB
. **Flash!**

ou avec **dd** (Linux) :

[source,bash]
----
# Identifier la cle USB
lsblk

# Flasher (REMPLACEZ /dev/sdX par votre device)
sudo dd if=magic_stick_0.1.8.iso of=/dev/sdX bs=4M status=progress && sync
----

[WARNING]
====
Cette operation **efface toutes les donnees** sur la cle USB. Verifiez le device avec `lsblk` avant.
====

=== Etape 2 : Booter sur la cle

1. Laissez la cle USB inseree
2. Redemarrez le PC
3. Accedez au BIOS/UEFI (**F2**, **F12** ou **Suppr**)
4. Selectionnez la cle USB dans le menu de boot
5. Le systeme live Xubuntu se lance

[NOTE]
====
Sur la plupart des PC recents, le menu de boot s'accede avec **F12**. Sur les PC plus anciens, essayez **F2** ou **Suppr**.
====

=== Etape 3 : Premier boot

Au premier demarrage :

* **Utilisateur** : `magic` (connexion automatique)
* **Clavier** : FR (AZERTY)
* **Locale** : fr_FR.UTF-8
* **Bureau** : Xubuntu XFCE

=== Etape 4 : Partition A/B (optionnel)

Pour configurer le partitionnement A/B avec persistence :

[source,bash]
----
sudo scripts/update-system.sh /dev/sdX
----

Voir la page link:ab-partition.html[Partition A/B] pour les details.
