=== 2. Diagnostic cause racine

* `arch.sh` `get_arch_suffix lazygit` → `Linux_x86_64` (L majuscule)
* `checksums.txt` sur GitHub Releases → `linux_x86_64` (l minuscule)
* `grep` dans le hook chroot est **case-sensitive** → ne matche pas → `EXPECTED` = ""
* `[ "" = "hash" ]` → FAIL

=== 3. Fix — `grep -i` case-insensitive

[source,diff]
----
- && LAZYGIT_EXPECTED=$(grep "lazygit_${LAZYGIT_VERSION#v}_${LAZYGIT_ARCH}.tar.gz" ...
+ && LAZYGIT_EXPECTED=$(grep -i "lazygit_${LAZYGIT_VERSION#v}_${LAZYGIT_ARCH}.tar.gz" ...
----

* Commit `442b0ba` push
* Build #25195816231 : **SUCCESS 0 warning**
  - lazygit checksum : plus aucun WARNING
  - ISO + Verify + Boot + Software + Mock A/B : tout vert
  - just checksum : deja OK
