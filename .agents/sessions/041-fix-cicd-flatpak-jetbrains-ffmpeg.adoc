== Leçons apprises

1. **Tar --strip-components et chemin final** : `tar -xzf file.tar.gz --strip-components=1` ne signifie PAS que le binaire se retrouve à la racine. Si l'archive contient `dir/bin/foo`, le résultat est `bin/foo`. Toujours vérifier le path réel avant de créer un symlink.
2. **Hook chroot "installed" ≠ fonctionnel** : Afficher `echo "installed"` ne vérifie pas la validité du symlink ni de l'exécutabilité. Ajouter `test -x /usr/local/bin/xxx` ou `readlink -e` dans le hook.
3. **apt-get autoremove est dangereux** : Tout paquet demandé explicitement dans `package-lists` mais installé comme dépendance d'un autre paquet peut être supprimé par autoremove. Protéger avec `apt-mark manual`.
