== Decisions

* **ISO live boot + installateur** : L'utilisateur veut ajouter ubiquity (installable en dur). Ajoute en EPIC 2, pas implemente cette session.
* **casper > live-boot** : Sur Ubuntu, casper est le standard. live-boot supprime des package-lists.
* **Hooks nettoyes** : Kernel/firmware installe par live-build, pas besoin de hook. GRUB bins dans le chroot = inutile.

== Lecons apprises

* Sur Ubuntu Noble, `exfat-utils` est obsolete → `exfatprogs`
* live-boot et casper sont incompatibles → garder uniquement casper
* `[SeatDefaults]` dans lightdm est deprecie → `[Seat:*]`
* Les hooks chroot s'executent dans le chroot de build, pas dans le systeme live
* install-software.sh doit tourner dans le live system, pas dans le chroot
