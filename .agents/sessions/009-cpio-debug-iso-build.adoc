== Analyse detaillee de la cause racine

L'erreur `cpio: premature end of archive` venait d'une cascade de problemes :

. **`--bootloader grub-efi`** : accepte par `lb config` mais **non gere** par les scripts `lb_binary_grub` (exit 0 si LB_BOOTLOADER != "grub"), `lb_binary_grub2` (ne gere que i386-pc), `lb_binary_iso` (cas `*` = unknown bootloader)
. **Pas de bootloader dans l'ISO** : `lb_binary_iso` tombe dans le cas par defaut et affiche un warning silencieux
. **`cpio` error** : vient du script `lb_binary_syslinux` qui tente de decomposer le `bootlogo` avec `cpio -i` mais le fichier n'existe pas car `gfxboot-theme-ubuntu` n'existe pas dans Ubuntu 24.04 (noble)
. **boot=live vs boot=casper** : le paquet `casper` est installe mais le param kernel dit `boot=live` (incompatibilite)

== Lecons apprises

* `--bootloader grub-efi` est accepte par `lb config` mais n'est gere par **aucun** script binaire live-build — c'est un piege
* `gfxboot-theme-ubuntu` a ete retire d'Ubuntu 24.04 (noble) — le theme `live-build` est l'alternative
* Les fichiers `includes.binary/` ecrasent les templates live-build sans avertissement — eviter de les utiliser pour les configs bootloader
* live-build en mode Ubuntu necessite `--initramfs casper` et `boot=casper`
* Le mode `iso-hybrid` necessite `syslinux` (pour `isohybrid`) dans l'environnement de build ET le chroot
