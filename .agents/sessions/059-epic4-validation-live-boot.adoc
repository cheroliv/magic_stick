- `config/live-build/includes.chroot/etc/systemd/system/smoke-test.service` : service conditionnel
  - `ConditionKernelCommandLine=smoke_test=true`
  - After=multi-user.target getty.target
  - TimeoutSec=300
- `scripts/test-boot.sh` : mode `--smoke`
  - Extrait kernel/initrd de l'ISO

  - Parse résultats, affiche PASS/FAIL
- `build.gradle.kts` : task `isoTestSmoke`
  - `./gradlew isoTestSmoke`

Approche : on utilise `-kernel` + `-initrd` + `-cdrom` pour passer `smoke_test=true` sur la ligne de commande noyau sans modifier l'ISO.
