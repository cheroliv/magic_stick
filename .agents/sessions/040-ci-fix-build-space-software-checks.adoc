== Leçons

. Les options `lb config` diffèrent entre les versions Debian. `--chroot-upgrades` n'existe pas dans `live-build` Ubuntu 24.04 ; utiliser le fichier `config/chroot_upgrades`.
. La step `Free disk space` classique des runners GitHub Actions ( suppression dotnet/android/ghc ) est nécessaire pour les builds ISO > 6 Go.
. Le cleanup post-build (`lb clean`) est essentiel pour libérer l'espace avant les tests qui extraient le squashfs.
