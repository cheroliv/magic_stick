== Lecons apprises

* live-build sur Ubuntu 24.04 necessite de nombreux patches : syslinux paths, rsvg-convert, casper paths, gfxboot-theme
* Les hooks live-build doivent avoir l'extension .chroot (pas .chroot.root)
* Le patching de scripts live-build par python3 est plus fiable que sed pour les patterns complexes
* isohybrid est dans syslinux-utils, pas syslinux
* lb_binary_iso nomme le fichier binary.hybrid.iso pour --binary-images iso-hybrid
* ISO hybride BIOS bootable generee avec succes — UEFI boot sera ajoute dans une prochaine session
